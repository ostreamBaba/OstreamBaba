## mybatis

* 持久化orm框架(内存的数据存入硬盘)

    * object relation mapping 对象关系映射


* 两个方面来说：

    * 大的方面，一个类对应表中的一行
    * 小的方面，一个类的属性对应一个表中的列


*   可以将java类转化成数据表的记录(jdbc)，也可以将数据表中的记录转化成java类(反射)


*   MyBatis是一种"半自动化"的orm映射框架(sql映射框架)
        * mybatis以sql语句为映射基础
*   hibernate是全自动化的orm映射框架
        * hibernate是彻底基于实体类与表进行映射
